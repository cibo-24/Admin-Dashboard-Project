# Admin-Dashboard-Project

Bu projede React, Material UI, Data Grid, Light ve Dark Mode kullanılmıştır.

Admin Dashboard içerisinde;

 - Nivo Charts
 - Formik
 - Yup
 - Full Calendar
 - Data Grid
 - 4 adet Veri Gösteren Chart
 - 3 adet Veri Tablosu
 - Sıkça Sorulan Sorular
 - Form Sayfası
 - Takvim Sayfası
